# 建材销售开单系统

独立版建材销售系统，包含手机号+密码登录、人员管理、客户/产品/订单展示、销售开单、退货单和销售单导出。

## 本地运行

```bash
npm start
```

默认地址：

```text
http://localhost:3000
```

## 当前数据存储

当前版本使用 `data/db.json` 作为轻量数据文件，方便先发布和验证完整业务流程。

正式商用建议升级为 PostgreSQL/MySQL 数据库，以支持多人长期稳定使用、备份和权限审计。
